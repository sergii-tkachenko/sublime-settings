# sublime-settings
My Sublime Text's User dir

```
rm -r ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/User
ln -nvfs ~/Development/settings/sublime-settings/User ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/User
ln -nvfs ~/Development/settings/sublime-settings/User/Preferences.home.sublime-settings ~/Development/settings/sublime-settings/User/Preferences.sublime-settings
```
